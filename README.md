Este repositório contém a interface front-end desenvolvida para um sistema de TCC que visa conectar os orientadores com os orientandos de forma mais simples, rápida e prática.
Para isso, foi utilizado HTML, CSS e JavaScript puro.
O projeto apresenta uma interface simples, responsiva e funcional, criada para demonstrar o fluxo e o design do sistema proposto no trabalho.

Tecnologias utilizadas
*HTML5 — estrutura da aplicação
*CSS3 — estilização e layout
*JavaScript — interatividade e lógica no cliente
*Imagens — elementos gráficos utilizados na interface

Funcionalidades implementadas
*Estrutura visual da aplicação
*Layout responsivo
*Elementos de navegação
*Componentes simples feitos em HTML/CSS
*Interações básicas em JavaScript (botões, eventos, validações, etc.)

Objetivo do projeto
Este front-end foi criado para demonstrar:
*A ideia geral do sistema
*O funcionamento básico da interface
*A proposta do layout
*O fluxo principal para o TCC
